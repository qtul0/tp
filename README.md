# SAFEA - Luxury Clothing Brand Website

A sophisticated, modern website for SAFEA, an elegant luxury clothing brand featuring black and purple color schemes.

## 🎨 Design Features

### Color Palette
- **Primary Black**: `#0f0f0f` - Deep, luxurious base
- **Secondary Black**: `#1a1a1a` - Background contrast
- **Dark Purple**: `#2d1b69` - Elegant primary accent
- **Medium Purple**: `#4a2a8f` - Secondary accent
- **Light Purple**: `#6b3fa0` - Highlight and hover effects
- **Gold Accent**: `#d4af37` - Premium luxury touch

### Typography
- **Headings**: Playfair Display (elegant serif)
- **Body**: Montserrat (modern sans-serif)

## 📋 Sections

1. **Navigation Bar** - Sticky navigation with smooth scroll links
2. **Hero Section** - Animated gradient background with call-to-action
3. **Featured Collection** - Product grid showcase with 4 luxury items
4. **About Section** - Brand story and core features
5. **Newsletter** - Email subscription form
6. **Contact** - Location, contact info, and social links
7. **Footer** - Copyright information

## ✨ Interactive Features

- Smooth scroll navigation
- Hover animations on collection items
- Parallax scrolling effect on hero
- Intersection Observer for fade-in animations
- Responsive newsletter form
- Dynamic notification system
- Mobile-responsive design

## 🚀 Getting Started

1. Save the files locally:
   - `index.html`
   - `styles.css`
   - `script.js`

2. Open `index.html` in your web browser

3. Customize the content:
   - Update brand information in the contact section
   - Modify collection items and pricing
   - Add your own brand images

## 📱 Responsive Design

- Desktop: Full layout with all features
- Tablet: Optimized grid layouts

## 🔧 Customization Guide

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-black: #0f0f0f;
    --accent-gold: #d4af37;
    /* ... other colors ... */
}
```

### Add Product Images
Replace the gradient backgrounds in collection items with real images:
```html
<div class="item-image" style="background-image: url('image.jpg'); background-size: cover;"></div>
```

### Update Contact Information
Modify the contact section with your actual details:
- Address
- Email
- Phone
- Social media links
